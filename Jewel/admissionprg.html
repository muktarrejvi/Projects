Engineering Admission Program

Features:-

  • There are 5-6 batches are available for first year student.
  • Each Batch Contains 50 students.
  • Ac Class Room.
  • Admission start after HSC exam.
  • Class Time 90 min/Batch.
  • Class Note Khata is provided from here.
  • Lecture book is provided from here.
  • Each lecture book contains a huge number of Engineering Universities admission test questions (MCQ + Written).
  • Every Student will get a chemistry guide book titled Chemistry Problem Solver.Which Contains a huge number of admission test question and their solution of Various Engineering,Medical,University and Science & Technology Universities.
  • For every Lecture book an exam (Written+MCQ in ORM sheet) is taken.
  • Exam result out very quickly.
  • Batches are devided into Two catagory.
    1. Saturday,Monday,Wednesday.
    2. Sunday,Tuesday,Thirsday.
  • Course Fee 6000/- Tk/student.
  • Batch time schedule are provided by Mobile SMS.

N.B: Course Fee Never Returnable.
